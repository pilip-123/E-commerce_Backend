# Backend structure scaffold (stubs)

This file documents the scaffold that was added to `Backend/E-Commerce_backend`:

- Admin controllers + stub views
- API controllers + stub routes
- Admin middleware stub

Run the Laravel server to verify routes:

- `cd Backend/E-Commerce_backend`
- `php artisan serve`

Then open:

- `http://127.0.0.1:8000/admin`

And test API endpoints:

- `http://127.0.0.1:8000/api/products`
